# 今天吃什么（MVP）

一个可直接部署到 Vercel 的静态网站 MVP，包含：

- 大转盘随机抽菜谱
- 每轮固定 8 道菜
- 一键重新选取本轮 8 道菜

## 本地预览

直接双击 `index.html` 即可打开，或使用任意静态服务器。

## 部署到 Vercel

1. 登录 [Vercel](https://vercel.com)
2. 点击 **Add New -> Project**
3. 选择你的代码仓库（或直接上传当前目录）
4. Framework Preset 选 **Other**（静态站点）
5. 保持默认设置并点击 Deploy

## 推荐域名

主域名建议：`chishenme.app`
